# Splunk configuration

Configuration reference for the **Splunk** data source (`grafana-splunk-datasource`) in Grafana.

For more information, see the [official documentation](https://grafana.com/docs/plugins/grafana-splunk-datasource).

> Generated from [`dsconfig.json`](dsconfig.json). Do not edit by hand — run `go generate ./...` to refresh.

## Provisioning examples

Each scenario below shows how to provision the data source in Grafana using a YAML file (loaded by Grafana's [file provisioner](https://grafana.com/docs/grafana/latest/administration/provisioning/#data-sources)) and using the [Grafana Terraform provider](https://registry.terraform.io/providers/grafana/grafana/latest/docs/resources/data_source).

Placeholders like `<YOUR_TOKEN>` must be replaced with real values before use.

### Basic authentication (`BasicAuth`)

**Grafana provisioning YAML**

```yaml
apiVersion: 1
datasources:
  - name: Splunk
    type: grafana-splunk-datasource
    access: proxy
    basicAuthUser: User
    url: URL
    jsonData:
      authType: BasicAuth
      fieldSearchType: quick
      internalFieldsFiltration: false
      maxResultCount: 0
      oauthPassThru: false
      pollSearchResult: false
      previewMode: false
      serverName: domain.example.com
      streamMode: false
      timeoutInSeconds: 30
      tlsAuth: false
      tlsAuthWithCACert: false
      tlsSkipVerify: false
      tsField: _time
      variableSearchLevel: fast
    secureJsonData:
      basicAuthPassword: "<YOUR_PASSWORD>"
      tlsCACert: "<YOUR_CA_CERTIFICATE>"
      tlsClientCert: "<YOUR_CLIENT_CERTIFICATE>"
      tlsClientKey: "<YOUR_CLIENT_KEY>"
```

**Terraform**

```hcl
resource "grafana_data_source" "grafana_splunk_datasource_BasicAuth" {
  type = "grafana-splunk-datasource"
  name = "Splunk"
  url = "URL"

  json_data_encoded = jsonencode({
    authType = "BasicAuth"
    fieldSearchType = "quick"
    internalFieldsFiltration = false
    maxResultCount = 0
    oauthPassThru = false
    pollSearchResult = false
    previewMode = false
    serverName = "domain.example.com"
    streamMode = false
    timeoutInSeconds = 30
    tlsAuth = false
    tlsAuthWithCACert = false
    tlsSkipVerify = false
    tsField = "_time"
    variableSearchLevel = "fast"
  })

  secure_json_data_encoded = jsonencode({
    basicAuthPassword = "<YOUR_PASSWORD>"
    tlsCACert = "<YOUR_CA_CERTIFICATE>"
    tlsClientCert = "<YOUR_CLIENT_CERTIFICATE>"
    tlsClientKey = "<YOUR_CLIENT_KEY>"
  })
}
```
