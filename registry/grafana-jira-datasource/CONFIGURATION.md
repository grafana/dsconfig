For more information, see the [official documentation](https://grafana.com/docs/plugins/grafana-jira-datasource).

## Connection

### Provider

_optional · radio_

The provider is where your JIRA instance is hosted. Select Jira Cloud if your Jira is being hosted on Atlassian Cloud. Select Jira Data Center / Jira Server if your Jira is being hosted elsewhere.

| | |
|---|---|
| Default | `cloud` |
| Allowed values | `cloud` (Jira Cloud), `server` (Jira Data Center / Jira Server) |
| Disabled when | **Authentication method** is **OAuth 2.0 — Service Account** (`oauth2`) |

### URL

_**required** · string_

The URL is the root URL for your Atlassian instance (example: company.atlassian.net).

| | |
|---|---|
| Example | `URL` |
