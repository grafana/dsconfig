# Elasticsearch configuration

Configuration reference for the **Elasticsearch** data source (`elasticsearch`) in Grafana.

For more information, see the [official documentation](https://grafana.com/docs/grafana/latest/datasources/elasticsearch/).

> Generated from [`dsconfig.json`](dsconfig.json). Do not edit by hand — run `go generate ./...` to refresh.

## Provisioning examples

Each scenario below shows how to provision the data source in Grafana using a YAML file (loaded by Grafana's [file provisioner](https://grafana.com/docs/grafana/latest/administration/provisioning/#data-sources)) and using the [Grafana Terraform provider](https://registry.terraform.io/providers/grafana/grafana/latest/docs/resources/data_source).

Placeholders like `<YOUR_TOKEN>` must be replaced with real values before use.

### Default configuration

**Grafana provisioning YAML**

```yaml
apiVersion: 1
datasources:
  - name: Elasticsearch
    type: elasticsearch
    access: proxy
    basicAuth: false
    basicAuthUser: User
    url: "http://localhost:9200"
    jsonData:
      apiKeyAuth: false
      defaultQueryMode: metrics
      includeFrozen: false
      index: es-index-name
      maxConcurrentShardRequests: 5
      oauthPassThru: false
      serverName: domain.example.com
      sigV4Auth: false
      timeField: "@timestamp"
      tlsAuth: false
      tlsAuthWithCACert: false
      tlsSkipVerify: false
    secureJsonData:
      apiKey: "<YOUR_API_KEY>"
      basicAuthPassword: "<YOUR_PASSWORD>"
      tlsCACert: "<YOUR_CA_CERTIFICATE>"
      tlsClientCert: "<YOUR_CLIENT_CERTIFICATE>"
      tlsClientKey: "<YOUR_CLIENT_KEY>"
```

**Terraform**

```hcl
resource "grafana_data_source" "elasticsearch" {
  type = "elasticsearch"
  name = "Elasticsearch"
  url = "http://localhost:9200"

  json_data_encoded = jsonencode({
    apiKeyAuth = false
    defaultQueryMode = "metrics"
    includeFrozen = false
    index = "es-index-name"
    maxConcurrentShardRequests = 5
    oauthPassThru = false
    serverName = "domain.example.com"
    sigV4Auth = false
    timeField = "@timestamp"
    tlsAuth = false
    tlsAuthWithCACert = false
    tlsSkipVerify = false
  })

  secure_json_data_encoded = jsonencode({
    apiKey = "<YOUR_API_KEY>"
    basicAuthPassword = "<YOUR_PASSWORD>"
    tlsCACert = "<YOUR_CA_CERTIFICATE>"
    tlsClientCert = "<YOUR_CLIENT_CERTIFICATE>"
    tlsClientKey = "<YOUR_CLIENT_KEY>"
  })
}
```
